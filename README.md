# Голосовая Диктовка

Приложение для голосовой диктовки с поддержкой русского и английского языков, разработанное с использованием C++ и Qt 6.

## Возможности

- Запись аудио с микрофона
- Распознавание речи на русском и английском языках
- Сохранение распознанного текста в файл
- Визуализация уровня громкости микрофона
- Система уведомлений о событиях

## Технологии

- C++17
- Qt 6 (Core, Widgets, Multimedia, Quick, QuickControls2)
- QML для современного пользовательского интерфейса

## Компоненты

- **MainWindow** - главное окно приложения с интерфейсом пользователя
- **SpeechRecognizer** - класс для записи и распознавания речи
- **QmlIntegration** - класс для связи C++ и QML компонентов
- **WaveProgress** - анимированная визуализация уровня микрофона (QML)
- **Notification** - компонент для отображения уведомлений (QML)
- **NotificationManager** - система управления уведомлениями (QML)

## Сборка и запуск

### Предварительные требования

- Qt 6.2 или выше
- CMake 3.14 или выше
- Компилятор с поддержкой C++17

### Инструкция по сборке

1. Клонировать репозиторий:
   ```
   git clone https://github.com/gqwg2003/voice_dictation.git
   cd voice_dictation
   ```

2. Создать директорию для сборки:
   ```
   mkdir build
   cd build
   ```

3. Сконфигурировать проект с помощью CMake:
   ```
   cmake ..
   ```

4. Собрать проект:
   ```
   cmake --build .
   ```

5. Запустить приложение:
   ```
   ./VoiceDictation
   ```

## Развертывание на Windows

Для создания автономного приложения на Windows можно использовать утилиту windeployqt:

```
windeployqt VoiceDictation.exe
```
