# Resources Directory

This directory contains resource files for the application, such as icons and images.

## Required Icons

The application uses the following icon files:

- app.png - Main application icon
- mic.png - Microphone icon for the record button
- copy.png - Copy icon for the clipboard button  
- clear.png - Clear icon for the clear button
- settings.png - Settings icon for the settings button
- help.png - Help icon for the help button

You can use any icons you prefer, just make sure they have the correct filenames.
Free icon resources:
- https://icons8.com/
- https://www.flaticon.com/
- https://material.io/resources/icons/
